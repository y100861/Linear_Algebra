# 선형방정식과 선형시스템

## Linear Equation(선형방정식)

- $a_{1}x_{1} + a_{2}x_{2} + ... + a_{n}x_{n} = b$
  - 위 식을 선형방정식으로 표현을 하면
  - $a^{T}x = b$ <br/>
