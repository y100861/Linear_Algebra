# 정규방정식(Normal Equation)

## 정규방정식(Normal Equation)

- $A^T(b - A\hat x) = 0$ <br/>
  $A^Tb - A^TA\hat x = 0$ <br/>
  $A^TA\hat x = A^Tb$

  - $\hat x$을 편미분한 값임
  - 미분을 하면서 상수는 0으로 없어지고, 다른 값들은 편미분을 하여 미분값이 만들어짐
  - $-A^Tb - A^Tb + 2A^TAx = 0$ <br/>
    $A^TAx = A^Tb$ <br/>
    $x = (A^TA)^-1A^Tb$
