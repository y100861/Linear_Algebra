# 대각화(Diagonalization)


## 대각화 행렬

- $D = V^-1AV$
- 대각화는 $A$행렬이 정사각 행렬일 때만을 따짐
- $D$가 존재할려면 $V$의 역행렬이 존재해야 함
  - 즉, V도 정사각 행렬이고 V의 각 벡터들은 선형독립 관계여야 함 <br/>

## V와 D를 찾는 과정

- $D = V^-1AV => VD = AV$
- $V$의 역행렬이 존재한다면 $VD = AV => D = V^-1AV$
